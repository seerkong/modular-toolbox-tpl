# Example 模块规则

本目录包含模块特定的规则。

## 规则

### RULE-EXAMPLE-001: 保持示例简洁

Example 模块的代码应该保持简洁易读，作为其他模块的参考。

### RULE-EXAMPLE-002: Vue 和 React 同步

Vue 和 React 的实现应该功能一致，便于开发者参考任一版本。
